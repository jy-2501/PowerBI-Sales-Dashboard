# 📊 Sales Performance Dashboard | Power BI

## Project Overview
Developed an interactive Sales Performance Dashboard in Power BI to monitor sales, profit, orders, customers, average order value, and profit margin. The dashboard enables users to analyze sales performance across regions, categories, products, payment modes, and salespersons.

## Dashboard Features
- KPI Cards
- Regional Sales Analysis
- Category-wise Sales Distribution
- Top Products Analysis
- Monthly Sales Trend
- Interactive Slicers
- Custom Tooltips
- Dynamic Business Insights
- DAX Measures
- Power Query Data Transformation

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

## Key Business Insights
- South region generated the highest sales.
- Furniture contributed the highest revenue.
- Monthly sales declined after June.
- Desk was the best-selling product.
- Average Order Value was 24.31K.
